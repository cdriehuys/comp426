# COMP 426

Assignments for COMP 426 at UNC.


## Development

[Browser Sync](https://browsersync.io/) is used to handle live-reloading in development. It is run with [gulp](https://gulpjs.com/). To get started:

```bash
$ yarn
$ yarn run server
```
